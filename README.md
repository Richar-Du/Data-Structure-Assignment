# 数据结构作业

本仓库实现了以二叉搜索树为基础的TreeSet和TreeMap，并进一步尝试将二叉搜索树延伸至红黑树实现。

学习资料：[小码哥《恋上数据结构与算法》笔记（十二）：集合（Set）与映射（Map）](https://juejin.cn/post/6844904071535542280#heading-13) ，并用python语言进行了重写。

  - Map
